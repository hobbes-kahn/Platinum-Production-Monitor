## 🇧🇷 Português
### 📋 Visão Geral
O **Platinum Monitor** é o módulo de coleta de dados do Projeto Platinum. Trata-se de uma aplicação desktop desenvolvida para automatizar o processo de amostragem em linhas de montagem, substituindo as pranchetas e cronômetros tradicionais. O sistema captura tempos de ciclo e interrupções em tempo real, economizando centenas de horas de engenheiros de processo (Green/Black Belts) e gestores.

### ⚙️ Funcionalidades Core
*   **Fator de Divisão Ajustável:** Permite que o operador estime a relação de peças necessárias para fechar um produto final, calculando KPIs e projeções de *Lead Time* automaticamente.
*   **Log de Interrupções:** Registro imediato de quebras de fluxo com categorização de motivos.
*   **Geração de Relatórios (TXT/CSV):** Exportação direta dos dados brutos e tempos produzidos para análise.
*   **Análise de Pareto Integrada:** Mapeamento automático dos principais ofensores de fluxo, direcionando os esforços de Melhoria Contínua.

### 🚀 Visão de Produto & Roadmap (O Oceano Azul)
Este aplicativo não é apenas um cronômetro digital; ele é a camada fundamental de coleta de dados para o desenvolvimento de um **Digital Twin (Gêmeo Digital)** acessível. O objetivo estratégico é levar tecnologia de alto nível de gestão para pequenas e médias empresas (PMEs) no Brasil, democratizando a eficiência industrial, reduzindo gargalos (via Teoria das Restrições) e impulsionando a qualificação do mercado nacional.

### 🛠 Tecnologias
*   **Python:** Lógica principal e interface gráfica (GUI).
*   **Manipulação de Dados:** Geração de logs e estruturação para o diagrama de Pareto.

## 🇺🇸 English
### 📋 Overview
The **Platinum Monitor** is the core data collection module of Project Platinum. It is a desktop application designed to automate the sampling process on assembly lines, replacing traditional clipboards and stopwatches. The system captures cycle times and interruptions in real-time, saving hundreds of hours for process engineers (Green/Black Belts) and management.

### ⚙️ Core Features
*   **Adjustable Division Factor:** Allows the operator to estimate the ratio of parts needed to complete a final product, automatically calculating KPIs and Lead Time projections.
*   **Interruption Logging:** Immediate recording of flow breaks with root-cause categorization.
*   **Report Generation (TXT/CSV):** Direct export of raw data and production times for analysis.
*   **Integrated Pareto Analysis:** Automatic mapping of the main flow offenders, directing Continuous Improvement efforts.

### 🚀 Product Vision & Roadmap (The Blue Ocean)
This application is not just a digital stopwatch; it is the foundational data collection layer for the development of an accessible **Digital Twin**. The strategic goal is to bring high-level management technology to Small and Medium Enterprises (SMEs) in Brazil, democratizing industrial efficiency, reducing bottlenecks (via Theory of Constraints), and boosting the qualification of the national market.

### 🛠 Stack
*   **Python:** Core logic and Graphical User Interface (GUI).
*   **Data Processing:** Log generation and data structuring for Pareto analysis.
